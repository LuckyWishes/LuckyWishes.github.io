# 插值操作
  1. Mustache语法 {{}} 双大括号（胡子 / 胡须）
  2. Mustache 语法中可以是变量和简单的表达式
  3. 插值操作–相关指令

    1. v-once 表示元素和组件只渲染一次，不会随着数据的改变而改变（无响应）

    2. v-html 该指令后面跟上一个 string 类型，会将 string 的 html 解析出来并且进行渲染

    3. v-text 使用较少，因为存在覆盖原有元素

    4. v-pre 用于跳过这个元素和它的子元素的编译过程，用于显示原本的 Mustache 语法

    5. v-cloak 在某些情况下，我们浏览器可能会显示出未编译的 Mustache 标签
