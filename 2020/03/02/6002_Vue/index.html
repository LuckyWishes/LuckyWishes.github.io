# 3. Vue 中 options 选项

  1. el

    1. 类型： String 或 DOM元素
    2. 作用：挂载 Vue 实例管理的 DOM 元素

  2. data

    1. 类型： Object 或 Function(组件当中data必须是一个函数)
    2. 作用：Vue 实例对应的数据对象

  3. methods

    1. 类型： {[key:sting]:Function}
    2. 作用：定义属于 Vue 的一些方法，可以在其他地方调用，可以在指令中使用

附加：方法 (method) 和函数 (Function) 的区别

  1. 一般实例对象里面的称为方法

  2. Vue 实例属性和方法，都有 $ 前缀，便于和用户定义的属性区别开
